# Stephen Portfolio

A clean, responsive one-page portfolio built with plain HTML, CSS, and JavaScript.

## Customize Before Applying

Replace these items in `index.html`:

- `Stephen` with your full name or professional brand.
- `tentaccion99@gmail.com` if you want to use a different real email.
- The hero headline and intro with your target role.
- The three project cards with real projects, measurable outcomes, and live links.
- The strengths section with the skills that match the job description.
- The contact section with your strongest achievement and preferred links.

## High-Impact Copy Formula

For each project, use this structure:

`Built [thing] for [audience], improving [metric] by [result] using [tools/approach].`

Examples:

- Built a responsive sales dashboard for operators, reducing weekly reporting time by 60% using React and charting APIs.
- Redesigned a checkout flow for mobile buyers, improving conversion by 18% through clearer pricing and trust signals.

## How To Open

Open `index.html` directly in your browser. No build step is required.

## Deploy On Vercel

This is a static site. On Vercel, use the project root as-is and leave the build
command empty. Vercel can serve `index.html` directly.

For the easiest GitHub upload, your repository can look like this with no
folders required:

```text
index.html
resume.html
styles.css
script.js
vercel.json
ai-workspace.svg
commerce-work.svg
dashboard-work.svg
portfolio-visual.svg
```

If images do not show on Vercel, open this URL pattern in your browser:

```text
https://your-site.vercel.app/dashboard-work.svg
```

If that gives a 404, the image file was not uploaded to GitHub at the same level
as `index.html`.
